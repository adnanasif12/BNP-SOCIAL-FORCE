# BNP Social Force - Backend Admin Panel

## 📋 সিস্টেম বর্ণনা

Backend Admin Panel একটি সম্পূর্ণ ডেটা ম্যানেজমেন্ট সিস্টেম যা সদস্যপদ এবং দান পরিচালনার জন্য তৈরি।

## 🔐 অ্যাক্সেস পদ্ধতি

### Admin Panel খোলা:
- পৃষ্ঠার নীচে-ডানে ⚙️ বোতাম ক্লিক করুন
- পাসওয়ার্ড দিয়ে লগইন করুন

### Default পাসওয়ার্ড:
```
admin123
```

**নোট:** পরবর্তীতে সার্ভার সাইড থেকে পাসওয়ার্ড ম্যানেজমেন্ট বাস্তবায়ন করতে হবে।

---

## 📊 ফিচার

### 1️⃣ সদস্যপদ ব্যবস্থাপনা (Member Form)

**ফিচার:**
- ✅ নতুন সদস্য যুক্ত করা
- ✅ সদস্যের তথ্য সংরক্ষণ (নাম, ইমেইল, ফোন, জেলা, থানা, সদস্যপদ প্রকার)
- ✅ সদস্য তালিকা দেখা
- ✅ সদস্য খুঁজা (নাম/ইমেইল/ফোন দ্বারা)
- ✅ সদস্য মুছে ফেলা
- ✅ সদস্যপঞ্জি এক্সেল ফাইলে ডাউনলোড করা

**ডাউনলোড ফরম্যাট:**
```
পূর্ণ নাম | ইমেইল | ফোন | জেলা | থানা | সদস্যপদ প্রকার | যোগদানের তারিখ
```

### 2️⃣ দান পেমেন্ট সিস্টেম (Donation Payment)

**ফিচার:**
- ✅ দান রেকর্ড করা
- ✅ বিভিন্ন পেমেন্ট পদ্ধতি (bKash, Nagad, Rocket, Bank Transfer, নগদ)
- ✅ লেনদেন আইডি ট্র্যাকিং
- ✅ দান উদ্দেশ্য শ্রেণিবদ্ধতা
- ✅ মোট দান, গণনা এবং গড় দান পরিসংখ্যান দেখা
- ✅ দান রেকর্ড খুঁজা এবং সম্পাদনা
- ✅ দান রেকর্ড এক্সেল ফাইলে ডাউনলোড করা

**ডাউনলোড ফরম্যাট:**
```
দাতার নাম | ইমেইল | ফোন | দান পরিমাণ | পেমেন্ট পদ্ধতি | লেনদেন আইডি | উদ্দেশ্য | তারিখ | মন্তব্য
```

## 🗂️ ফাইল স্ট্রাকচার

```
src/backend/
├── components/
│   ├── AdminPanel.jsx        # Admin Panel লগইন & কন্টেইনার
│   ├── Backend.jsx           # মেইন ড্যাশবোর্ড
│   ├── MemberForm.jsx        # সদস্য ফর্ম ও ম্যানেজমেন্ট
│   └── DonationPayment.jsx   # দান পেমেন্ট সিস্টেম
├── styles/
│   ├── AdminPanel.css        # Admin Panel স্টাইল
│   ├── Backend.css           # ড্যাশবোর্ড স্টাইল
│   ├── MemberForm.css        # সদস্য ফর্ম স্টাইল
│   └── DonationPayment.css   # দান পেমেন্ট স্টাইল
├── utils/
│   └── excelExport.js        # এক্সেল ডাউনলোড ফাংশন
└── index.js                  # এক্সপোর্ট ফাইল
```

## 📥 এক্সেল ডাউনলোড সেটআপ

### বিকল্প ১: CSV ফরম্যাট (ডিফল্ট)
- কোন অতিরিক্ত লাইব্রেরি প্রয়োজন নেই
- সব ব্রাউজারে কাজ করে

### বিকল্প ২: XLSX ফরম্যাট (অপ্টিমাল)
প্রকৃত Excel ফাইলের জন্য, SheetJS ইনস্টল করুন:

```bash
npm install xlsx
```

তারপর এক্সেল এক্সপোর্ট ব্যবহার করুন:
```javascript
import { downloadExcelAdvanced } from './backend/utils/excelExport';
downloadExcelAdvanced(data, 'filename');
```
