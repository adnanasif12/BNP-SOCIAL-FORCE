## 🎯 তাৎক্ষণিক শুরু করুন (5 মিনিট)

### পদক্ষেপ ১: ডিপেন্ডেন্সি ইনস্টল করুন
```bash
npm install
```

### পদক্ষেপ २: পরিবেশ সেটআপ করুন
`.env.local` ফাইল সম্পাদনা করুন এবং যোগ করুন:
```env
MONGODB_URI=mongodb+srv://admin:password@cluster.mongodb.net/bnp_socialforce
JWT_SECRET=your-secret-key
API_BASE_URL=http://localhost:5173
NODE_ENV=development
```

### পদক্ষেপ ३: Admin তৈরি করুন
```bash
npm run setup-admin
```

### পদক্ষেপ ४: সার্ভার চালান
```bash
npm run dev
```

✅ সম্পূর্ণ! আপনার API চলছে এ: `http://localhost:5173/api`

## 🔗 API এন্ডপয়েন্ট

### প্রমাণীকরণ
- `POST /api/auth/login` - প্রশাসক লগইন
- `POST /api/auth/register` - নতুন প্রশাসক

### সদস্য ব্যবস্থাপনা
- `GET /api/members` - সদস্যদের তালিকা
- `POST /api/members` - নতুন সদস্য যোগ করুন
- `PUT /api/members?id=:id` - সদস্য আপডেট করুন
- `DELETE /api/members?id=:id` - সদস্য মুছে ফেলুন

### দান ব্যবস্থাপনা
- `GET /api/donations` - দান তালিকা
- `POST /api/donations` - নতুন দান যোগ করুন
- `PUT /api/donations?id=:id` - দান আপডেট করুন
- `DELETE /api/donations?id=:id` - দান মুছে ফেলুন

### ড্যাশবোর্ড
- `GET /api/dashboard` - পরিসংখ্যান এবং বিশ্লেষণ
