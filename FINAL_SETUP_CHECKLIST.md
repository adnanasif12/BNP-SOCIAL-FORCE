## 📦 পর্যায় ১: স্থানীয় ডেভেলপমেন্ট সেটআপ

### ১.१ নোড ডিপেন্ডেন্সি ইনস্টল করুন
```bash
npm install
```
- [ ] সব প্যাকেজ সফলভাবে ইনস্টল হয়েছে
- [ ] `node_modules` ফোল্ডার তৈরি হয়েছে
- [ ] কোন ত্রুটি নেই

### १.२ পরিবেশ ভেরিয়েবল সেটআপ করুন

**`.env.local` ফাইল চেক করুন:**

```bash
# এই ফাইলটি ইতিমধ্যে রয়েছে
# শুধু আপনার মূল্যবোধ দিয়ে পূরণ করুন
```

**প্রয়োজনীয় ভেরিয়েবল:**
- [ ] `MONGODB_URI` - আপনার MongoDB সংযোগ স্ট্রিং
- [ ] `JWT_SECRET` - আপনার JWT সিক্রেট কী
- [ ] `API_BASE_URL` - আপনার API বেস URL
- [ ] `NODE_ENV` - `development` অথবা `production`

### १.३ MongoDB Atlas ক্লাস্টার সেটআপ করুন

1. [mongodb.com/cloud/atlas](https://mongodb.com/cloud/atlas) এ যান
2. সাইন আপ বা লগইন করুন
3. **Create Database** ক্লিক করুন:
   - [ ] ক্লাস্টার নাম: `bnp-social-force`
   - [ ] প্ল্যান: **Free** (M0)
   - [ ] ক্লাউড প্রোভাইডার: AWS
   - [ ] রিজিয়ন: Asia Pacific (সিঙ্গাপুর)

4. **Create a database user:**
   - [ ] Username: `admin`
   - [ ] Password: একটি শক্তিশালী পাসওয়ার্ড

5. **IP Whitelist:**
   - [ ] `0.0.0.0/0` যোগ করুন (সব IP থেকে অ্যাক্সেস অনুমতি)

6. **সংযোগ স্ট্রিং পান:**
   - [ ] **Connect** ক্লিক করুন
   - [ ] **Connect your application** নির্বাচন করুন
   - [ ] সংযোগ স্ট্রিং কপি করুন
   - [ ] `.env.local` এ `MONGODB_URI` এ পেস্ট করুন

## 🌐 পর্যায় ४: Vercel ডিপ্লয়মেন্ট

### ४.१ Vercel অ্যাকাউন্ট সেটআপ করুন

1. [vercel.com](https://vercel.com) এ যান
2. **Sign Up** ক্লিক করুন
3. **GitHub দিয়ে সাইন আপ করুন** নির্বাচন করুন
4. GitHub অ্যাক্সেস অনুমতি দিন

- [ ] Vercel অ্যাকাউন্ট তৈরি হয়েছে
- [ ] GitHub সংযুক্ত হয়েছে

### ४.२ প্রজেক্ট Vercel এ ইম্পোর্ট করুন

1. Vercel ড্যাশবোর্ডে **Add New** → **Project** ক্লিক করুন
2. আপনার GitHub রিপোজিটরি নির্বাচন করুন
3. **Import** ক্লিক করুন

**কনফিগারেশন সেট করুন:**
- [ ] Framework: Vite
- [ ] Build Command: `npm run build`
- [ ] Output Directory: `dist`
- [ ] Install Command: `npm ci`

### ४.३ Environment ভেরিয়েবল যোগ করুন

Vercel এ **Settings** → **Environment Variables** যান:

```
MONGODB_URI = mongodb+srv://admin:password@cluster.mongodb.net/bnp_socialforce?retryWrites=true&w=majority
JWT_SECRET = (আপনার JWT সিক্রেট কী)
API_BASE_URL = https://your-project-name.vercel.app
NODE_ENV = production
```

- [ ] সব ভেরিয়েবল সেট করা হয়েছে
- [ ] মান সঠিক

### ४.४ Vercel এ ডিপ্লয় করুন

2. ডিপ্লয়মেন্ট সম্পন্ন হওয়ার জন্য অপেক্ষা করুন (2-5 মিনিট)
3. **Visit** ক্লিক করুন

- [ ] ডিপ্লয়মেন্ট সফল হয়েছে
- [ ] প্রজেক্ট URL প্রাপ্ত হয়েছে (যেমন: `https://bnp-social-force.vercel.app`)
- [ ] ওয়েবসাইট অ্যাক্সেসযোগ্য

### ५.३ DNS রেকর্ড কনফিগার করুন

**যদি GoDaddy ব্যবহার করেন:**

1. GoDaddy এ লগইন করুন
2. **My Products** → আপনার ডোমেইন → **DNS** যান
3. নতুন রেকর্ড যোগ করুন:

| Type | Name | Value | TTL |
|------|------|-------|-----|
| CNAME | www | `cname.vercel.com` | 3600 |
| A | @ | `76.76.19.132` | 3600 |

- [ ] CNAME রেকর্ড যোগ করা হয়েছে
- [ ] A রেকর্ড যোগ করা হয়েছে

**যদি Namecheap ব্যবহার করেন:**

1. Namecheap এ লগইন করুন
2. **Domain List** → আপনার ডোমেইন → **Manage** যান
3. **Advanced DNS** ট্যাব খুলুন
4. রেকর্ড যোগ করুন (উপরের মতো)

- [ ] DNS রেকর্ড যোগ করা হয়েছে
- [ ] মূল্যবোধ সঠিক
